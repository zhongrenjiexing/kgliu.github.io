# Personal Intro Page

This repository hosts a simple personal introduction webpage for GitHub Pages.

## Files

- `index.html`: Main personal intro page.

## Local Preview

You can open `index.html` directly in your browser, or run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## GitHub Pages

The site is deployed from the `main` branch root (`/`).
